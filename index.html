Prework Study Guide

HTML

  • Head is the container for Meta Data and usually found up top of code.
  • Meta are elements containing information for the browser about the page.
  • Elements refer to the entire element from open to close tags.
  • Tag is only what is inside the brackets.
  • Body is the main meat of the webpage. Often refered to as a parent element with child elements adding to the page within it.

CSS

  • Stands for Cascading Style Sheets.
  • Used as presentation and for rendering.
  • atrisk is used for applying to all visual elements
  • To apply mulptiple selectors, use a coma to sperate each element or attribute.

Git

  • It is an open source version control system for projects.
  • Used to track changes in source code.
  • Makes working with multiple developers possible.

JavaScript

  • Core technology of the internet.
  • Used to build logic and interactivity into the page.
  • First appeared in 1995.
